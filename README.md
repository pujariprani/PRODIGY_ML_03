Task 3: Cat and Dog Image Classification using Support Vector Machine (SVM)

Objective

The objective of this project is to build a Support Vector Machine (SVM) model to classify images of cats and dogs. The model is trained using image data and predicts whether a given image belongs to a cat or a dog.

Dataset

- Dataset: Kaggle Cats and Dogs Dataset
- Classes:
  - Cat
  - Dog

Tools and Libraries

- Python
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

Project Workflow

1. Load cat and dog images from the dataset.
2. Convert images to grayscale.
3. Resize images to 64×64 pixels.
4. Flatten images into feature vectors.
5. Split the dataset into training and testing sets.
6. Train the Support Vector Machine (SVM) model.
7. Predict labels for test images.
8. Evaluate model performance using accuracy score and confusion matrix.
9. Display sample predictions.

Features

- Image preprocessing using OpenCV.
- Binary image classification (Cat vs Dog).
- SVM model training and prediction.
- Accuracy evaluation.
- Visualization of sample predictions.

Output

- Classification of images as Cat or Dog.
- Accuracy score.
- Confusion Matrix.
- Sample prediction images.

Conclusion

This project successfully implements a Support Vector Machine (SVM) for image classification. The model is capable of distinguishing between cat and dog images with good accuracy after preprocessing and training.
